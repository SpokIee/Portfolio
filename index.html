GabAIRa Care

GabayRa Care: An AI-Driven Medication Management Application with IoT Pillbox Integration for Improved Adherence

AI-driven medication management app with IoT pillbox integration, built to help patients with chronic conditions (hypertension, diabetes) stay on top of their medication, with real-time caregiver updates. Uses an XGBoost model trained on 10,000 patient records to predict adherence risk and time reminders (send now / delay / snooze). Syncs with an ESP32-based smart pillbox over REST API — patients confirm doses via physical button or in-app "Taken" tap, caregivers see it live on their dashboard. Built with React Native (Expo) + Neon PostgreSQL. UAT with 9 respondents (incl. a pharmacist): 100% pass rate (Patient role), 99.1% (Caretaker role), ISO/IEC 25010 rating 4.09 ("Acceptable"), highest in Usability (4.80).

Team: Raphael Miguel P. Dumdum, Jan Jeerzyll T. Hudes, Andre Jan M. Pangan
